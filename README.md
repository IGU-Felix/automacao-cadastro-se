Este projeto está livre de dados sensíveis. Por esse motivo, pode não ser possível executá-lo corretamente sem as devidas configurações e dados 
de ambiente.

Além disso, a etapa final do projeto não foi totalmente concluída. O código realiza o processamento e cadastro de todos os dados a partir de 
uma planilha Excel, porém não foi configurado para execução automatizada.

O projeto foi desenvolvido como parte das minhas atividades como estagiário, incluindo a implementação de algumas melhorias, como a execução 
em modo headless.
